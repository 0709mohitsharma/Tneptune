# 🚀 PeterBrowser  
### A lightweight, distraction-free trading browser for Android  

> Built for traders who value **focus, speed, and reliability**

---

## ⚡ The Problem

Modern trading apps are cluttered.

Platforms like Dhan overload small screens with:
- Ads, news feeds, notifications  
- UI elements that shrink chart space  

Plus, they often suffer from:
- ❌ Network drops when screen turns off  
- ❌ Background data getting killed  
- ❌ No reliable real-time updates  
- ❌ Poor multi-chart usability  

---

## ✅ The Solution

**PeterBrowser** strips everything down to what matters:

- 🧘 **Zero clutter** → Full-screen charts only  
- 🔌 **Stable connection** → Keeps data flowing even when screen is off  
- 🔄 **True background mode** → Works like a persistent trading terminal  
- ⏱ **No screen timeout** → Stays active while you trade  
- ⚙️ **Custom platform support** → Load any trading URL  

---

## ✨ Features

- Fullscreen WebView overlay  
- Persistent foreground service  
- Background data connectivity  
- Keep-awake screen mode  
- Configurable trading URL  
- Optimized for real-time charts  
- Smart battery usage  

---

## 📱 Requirements

- Android 12+ (API 31+)  
- Overlay permission  
- Storage permission  

---

## 🛠 Setup

1. Install the APK  
2. Grant required permissions  
3. By default it's tv.dhan.co but you can edit trading_url in config file at /Downloads/Tneptune/config.txt to set and customise the url.
4. Restart the app
